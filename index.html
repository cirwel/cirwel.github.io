Kenny Wang —
runtime governance for
AI-agent fleets.

I build runtime safety infrastructure for autonomous AI agents — the layer that operates after deployment, while agents are actually running.

My framework, UNITARES, gives agents continuous awareness of their own operational state — coherence, entropy, calibration, drift — from behavioral telemetry and a dynamical-systems state model. It has run continuously on my own development fleet since November 2025. This page is the index of that work: papers, systems, datasets, and decks.
